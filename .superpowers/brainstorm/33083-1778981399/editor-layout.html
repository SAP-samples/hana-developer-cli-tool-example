Key Layout Decisions

  • Left panel (180px): Draggable node palette — drag onto canvas to add nodes
  • Center: Vue Flow canvas — DAG with bottom-to-top flow, dot-grid background, minimap, zoom controls
  • Right panel (320px): Context-sensitive properties — tabs change based on selected node type
  • Tab bar (top): Multiple open Calculation Views, like browser tabs
  • Toolbar: Undo/Redo, layout controls, save, toggle XML source view
  • Resizable panels: Using splitpanes (already in project) for left/right panel resize
