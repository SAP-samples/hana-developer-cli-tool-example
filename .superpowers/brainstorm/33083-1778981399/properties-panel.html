Interaction Model

  • Drag & Drop mapping: Drag from data source columns (left/middle) → drop onto Output Columns area (right). Mapped columns show strikethrough + "→ mapped" in source.
  • Join condition: Drag column from left source onto column from right source to create a join condition. Or click "+ Add Condition" and pick from dropdowns.
  • Column indicators: ◆ = attribute (dimension), ◇ = measure. Color-coded left border shows source origin.
  • Expression editor: Embedded Monaco editor with SQL syntax highlighting and column name autocomplete from available inputs.
  • Multi-select: Shift+click or Ctrl+click multiple columns to drag-map in bulk.
  • "Map All" shortcut: Double-click a data source header to map all its unmapped columns at once.
