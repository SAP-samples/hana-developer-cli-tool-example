Visual Language Rules

  • Color coding: Each node type has a unique accent color used for borders, icons, and handles
  • Handles: Bottom = inputs (from children), Top = output (to parent). Count matches actual connections.
  • Semantics: Special dashed border + gradient fill — always fixed at top of graph
  • Info density: Node name, type-specific subtitle, input/column count — scannable at a glance
  • Selected state: Glow shadow + handles become larger (like BAS blue circles)
  • Collapsed state: Reduced height, hides subtitle, shows only name + icon
