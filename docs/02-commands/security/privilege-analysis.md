# privilegeAnalysis

> Command: `privilegeAnalysis`  
> Category: **System Tools**  
> Status: Production Ready

## Description

Analyze user privileges and suggest least privilege

## Syntax

```bash
hana-cli privilegeAnalysis [options]
```

## Aliases

- `privanalysis`
- `privanalyze`

## Parameters

For a complete list of parameters and options, use:

```bash
hana-cli privilegeAnalysis --help
```

## Examples

### Basic Usage

```bash
hana-cli hana-cli privilegeAnalysis --user TESTUSER --suggest
```
