# activateHDI

> Command: `activateHDI`  
> Category: **HDI Management**  
> Status: Production Ready

## Description

Activate the HDI service in a particluar SAP HANA Tenant (Must be ran in the SYSTEMDB)

## Syntax

```bash
hana-cli activateHDI [tenant] [options]
```

## Aliases

- `ahdi`
- `ah`

## Parameters

For a complete list of parameters and options, use:

```bash
hana-cli activateHDI --help
```

## Examples

### Basic Usage

```bash
hana-cli hana-cli activateHDI --container hdi_container
```

Execute the command
