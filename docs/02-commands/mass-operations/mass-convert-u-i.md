# massConvertUI

> Command: `massConvertUI`  
> Category: **Mass Operations**  
> Status: Production Ready

## Description

Convert a group of tables to CDS or HDBTable format via browser based UI

## Syntax

```bash
hana-cli massConvertUI [schema] [table] [options]
```
