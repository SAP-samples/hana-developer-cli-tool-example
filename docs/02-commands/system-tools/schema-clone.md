# schemaClone

> Command: `schemaClone`  
> Category: **System Tools**  
> Status: Production Ready

## Description

Clone entire schema with or without data

## Syntax

```bash
hana-cli schemaClone [options]
```

## Aliases

- `schemaclone`
- `cloneSchema`
- `copyschema`

## Parameters

For a complete list of parameters and options, use:

```bash
hana-cli schemaClone --help
```

## Examples

### Basic Usage

```bash
hana-cli hana-cli schemaClone --sourceSchema SOURCE --targetSchema TARGET --includeData
```

Clone entire schema with or without data
