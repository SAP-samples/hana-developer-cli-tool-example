# iniContents

> Command: `iniContents`  
> Category: **System Tools**  
> Status: Production Ready

## Description

Contents of INI Configuration (filtered by File Name)

## Syntax

```bash
hana-cli iniContents [file] [section] [options]
```

## Aliases

- `if`
- `inifiles`

## Parameters

For a complete list of parameters and options, use:

```bash
hana-cli iniContents --help
```

## Examples

### Basic Usage

```bash
hana-cli hana-cli iniContents --file myFile
```
