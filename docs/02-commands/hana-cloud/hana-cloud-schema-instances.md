# hanaCloudSchemaInstances

> Command: `hanaCloudSchemaInstances`  
> Category: **HANA Cloud**  
> Status: Production Ready

## Description

List all SAP HANA Cloud Schema service instances in your target Space

## Syntax

```bash
hana-cli schemaInstances [options]
```

## Aliases

- `schemaServices`
- `listschemas`
- `schemaservices`

```bash

```

