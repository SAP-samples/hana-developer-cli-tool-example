# hanaCloudSBSSInstances

> Command: `hanaCloudSBSSInstances`  
> Category: **HANA Cloud**  
> Status: Production Ready

## Description

List all SAP HANA Cloud SBSS service instances in your target Space

## Syntax

```bash
hana-cli sbss [options]
```
