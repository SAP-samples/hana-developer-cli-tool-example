# helpDocu

> Command: `helpDocu`  
> Category: **Developer Tools**  
> Status: Production Ready

## Description

Open Help Documentation in web browser

## Syntax

```bash
hana-cli helpDocu [options]
```

## Aliases

- `openDocu`
- `openDocumentation`
- `documentation`
- `docs`
- `docu`

## Parameters

For a complete list of parameters and options, use:

```bash
hana-cli helpDocu --help
```

## Examples

### Basic Usage

```bash
hana-cli helpDocu
```

Execute the command to open the documentation site in your default web browser.

### Using Aliases

All of the following will open the same documentation:

```bash
hana-cli docs
hana-cli documentation
hana-cli openDocu
``` 

## Documentation Site

This command opens the SAP HANA Developer CLI documentation site at:
https://sap-samples.github.io/hana-developer-cli-tool-example/
