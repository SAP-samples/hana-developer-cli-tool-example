Maximum file size to allow for import (in megabytes)

Sets the size limit for source files. Files larger than this limit will be rejected to prevent memory issues and system overload.

Recommended Values:

  • Development/Testing: 100-500 MB
  • Production: 500-2000 MB (depending on system capacity)

Default: 500 MB

Note: This limit protects against accidental large file imports that could consume system resources.
