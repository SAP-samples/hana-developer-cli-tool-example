Number of rows to process in each batch

Batch size controls how many rows are processed together during import. Larger batches are faster but consume more memory. Smaller batches are slower but use less memory.

Recommended Values:

  • Small datasets (<10,000 rows): 5000-10000
  • Medium datasets (10K-1M rows): 1000-5000
  • Large datasets (>1M rows): 100-1000

Default: 1000

Performance Tip: Increase batch size for faster imports on systems with sufficient memory.
