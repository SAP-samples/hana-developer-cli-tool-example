Maximum time to allow for import process (in seconds)

Sets a timeout limit for the entire import operation. If the import takes longer than this time, it will be automatically terminated to prevent indefinite hanging.

Recommended Values:

  • Small datasets: 60-300 seconds
  • Medium datasets: 300-1800 seconds (5-30 minutes)
  • Large datasets: 1800-3600 seconds (30-60 minutes)

Default: 3600 seconds (1 hour)
