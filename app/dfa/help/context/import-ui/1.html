General information related to this screen.

The Import UI allows you to import data from Excel, CSV, or other file formats into SAP HANA tables. This screen provides various options for controlling how data is imported, including file format selection, column matching strategies, batch processing, and error handling.

Key Features:

  • Support for multiple file formats (CSV, Excel)
  • Flexible column matching (automatic, manual, strict)
  • Batch import processing for large files
  • Data validation and error handling
  • Dry-run mode for testing before actual import
  • Null value configuration
  • Performance tuning options
