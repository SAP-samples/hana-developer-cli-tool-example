General information related to this screen

Welcome to hana-cli Query Simple tool!

This screen provides a simple interface for executing SQL queries against your SAP HANA database and optionally exporting the results to various file formats. You can write and execute SELECT statements, view results in a table, and export data to JSON, Excel, CSV, or other formats.

Use this tool for ad-hoc queries, data extraction, quick database exploration, or exporting query results for reporting and analysis purposes.
