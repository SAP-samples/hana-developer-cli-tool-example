Output Format Selection

Select the format for exporting query results. Available options include:

Table - Display results only in the UI table below without creating a file.
JSON - Export as JavaScript Object Notation, useful for web applications and data interchange.
Excel - Export as Microsoft Excel format (.xlsx) for spreadsheet analysis.
CSV - Export as Comma Separated Values, universal format for data exchange.

Choose the format that best suits your needs for working with the query results. The Table option is ideal for quick viewing, while file formats enable further processing in other tools.
