Enable Debug Output

Enable this option to activate debug mode, which provides detailed diagnostic information, execution logs, and troubleshooting data. This is useful for investigating issues or understanding the internal operations of the query execution.

When debug mode is enabled, you'll see additional information such as:

  • Query execution plans and optimization details
  • Connection details and parameters
  • Timing information for query execution
  • Internal processing steps and data transformations
  • Any warnings or informational messages

Note: Debug mode is primarily intended for development and troubleshooting. It's recommended to disable it during normal operation as it may produce a large amount of output.
