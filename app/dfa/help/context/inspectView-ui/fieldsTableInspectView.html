View Column Listing

This table displays detailed information about all columns (fields) in the inspected view. The Fields tab provides a comprehensive view of the view's structure, including column names, data types, lengths, and other metadata.

Each row in this table represents a column in your database view. You can see information such as:

  • Column name and position
  • Data type (VARCHAR, INTEGER, DATE, etc.)
  • Length or precision for applicable data types
  • Nullable flag (whether the column allows NULL values)
  • Source table and column (for simple views)
  • Comments and column descriptions

You can select multiple rows to copy column information or use this view to understand the complete structure of your view's output.
