Basic Table Information

The Basic Info tab displays fundamental metadata about the selected table. This includes the table name, schema name, table type (e.g., COLUMN, ROW), and the table's object identifier (OID).

Additional information shown includes whether the table has a primary key, if data is preloaded into memory, the unload priority for memory management, and the table creation timestamp. This information helps you understand the table's configuration and characteristics at a glance.
