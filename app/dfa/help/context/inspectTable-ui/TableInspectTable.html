Target Table to Inspect

Enter the name of the table you want to inspect. You can type the table name directly or use the suggestion feature to search for available tables in the selected schema.

The table name is case-sensitive. If you're unsure of the exact name, start typing and suggestions will appear showing matching table names from your database schema. Once you select or enter a table name and execute the inspection, the tool will retrieve and display comprehensive metadata about the table structure.
