Target Schema Selection

Select the database schema that contains the table you want to inspect. You can type the schema name directly or use the suggestion feature to search for available schemas in your database.

Use the special value **CURRENT_SCHEMA** to automatically use your current connection's default schema. When this option is selected, the actual schema name will be displayed in the read-only field below.

The schema name is case-sensitive. Start typing to see suggestions of available schemas from your database connection.
