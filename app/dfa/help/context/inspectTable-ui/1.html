General information related to this screen

Welcome to hana-cli Inspect Table tool!

This screen allows you to inspect detailed metadata and information about SAP HANA database tables. You can view table structure, column definitions, constraints, and generate table definitions in various formats including SQL DDL, CDS, and HDBTable.

Use this tool to understand table schemas, verify table configurations, or extract table definitions for migration or documentation purposes.
